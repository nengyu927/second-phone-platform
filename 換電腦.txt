【第一次使用新電腦】
1. git clone <repo網址>
2. cd second-phone-platform
3. .\scripts\setup.ps1
   └─ 腳本會自動建立 .env，並詢問 MySQL root 密碼與管理員密碼
   └─ 若略過密碼，事後可直接編輯根目錄的 .env
4. 確認 .env 內容正確後，執行 .\scripts\start-all.ps1

注意：.env 不會提交 Git，每台電腦必須執行一次 setup.ps1

【每天開始製作】
git pull → 執行 .\scripts\start-all.ps1

【每天製作完成】
git add → git commit → git push
